# Сетевой анализ персонажей «Перси Джексона»

Учебный проект по Digital Humanities. Исследование сравнивает сети совместных упоминаний персонажей в первых трёх книгах цикла Рика Риордана.

## Исследовательский вопрос

Как меняется структура сети совместных упоминаний персонажей в первых трёх книгах и какие персонажи занимают центральные и посреднические позиции в каждой книге?

## Метод

- узлы — персонажи;
- связь — совместное появление в окне ±5 нормализованных строк;
- вес — суммарное число совместных появлений;
- граф — неориентированный и взвешенный;
- метрики — число узлов и рёбер, плотность, degree centrality, betweenness centrality;
- визуализация — PyVis, размер узла зависит от degree centrality, толщина ребра равна `weight ** 2.0`.

Метод соответствует логике учебной тетрадки по сетевому анализу.

## Данные

Источник: https://www.kaggle.com/datasets/shobhit043/percy-jackson-first-5-books

В репозиторий полные тексты тоже загружаются, но это на всякий.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook notebooks/percy_network_analysis.ipynb
```

Для сайта Quarto:

```bash
quarto preview
```

Для сборки сайта:

```bash
quarto render
```

## Структура

- `index.qmd` — главная страница;
- `analysis.qmd` — исследовательский текст;
- `notebooks/percy_network_analysis.ipynb` — полный воспроизводимый код;
- `data/characters.csv` — словарь персонажей и вариантов имён;
- `data/processed/` — таблицы рёбер;
- `results/tables/` — метрики;
- `results/figures/` — статичные графики;
- `results/networks/` — интерактивные сети.
